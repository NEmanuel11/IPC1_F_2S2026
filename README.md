-------Inicializacion del Programa------
1. Ingresar la opcion, brindada por el menú.
2. En caso que sea CREAR TABLERO
3. Ingrese tu nombre
4. Tipo de tablero; Pequeños, grande
5. Ingrese la cantidad de premios, fantasmas y paredes.
6. Movimiento 1, posicionar el personaje
7. Menú de diferentes movimientos, según la elección del usuario.
8. Tienes 3 vidas, en caso que las pierdas, el programa saldra.
9. Menú principal para ver la cantidad de puntos.
10. Menú principal salir del juegos. Terminara el Programa
